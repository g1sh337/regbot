# 📊 Бот учёта регистраций

## Установка и запуск

### 1. Получи токен бота
1. Открой Telegram, найди @BotFather
2. Отправь `/newbot`
3. Придумай имя и username для бота
4. Скопируй токен (выглядит как `123456789:ABCdef...`)

### 2. Вставь токен в config.py
Открой файл `config.py` и замени:
```
BOT_TOKEN = "ВАШ_ТОКЕН_СЮДА"
```
на твой реальный токен.

### 3. Установи Python и зависимости
```bash
# Убедись что Python 3.10+ установлен
python --version

# Установи библиотеки
pip install -r requirements.txt
```

### 4. Запусти бота
```bash
python bot.py
```

### 5. Добавь бота в чат
1. Открой настройки чата → Администраторы → Добавить администратора
2. Найди своего бота по username
3. Выдай права: **Чтение сообщений** (обязательно!)

## Как это работает

### Формат сообщения участника
Каждый участник пишет (или редактирует) своё сообщение в таком формате:
```
14.05.26
Рашид
мобилка казино 🇷🇺 россия - 5
мобилка казино(2) 🇷🇺 россия - 10
мобилка казино 🇦🇷 аргентина - 13
```

Бот автоматически суммирует все числа после ` - ` и обновляет сводку.

### Что делает бот
- При каждом новом сообщении или редактировании от участника — отправляет новую сводку в чат
- Сводка привязана к дате из сообщения (строка `14.05.26`)
- Данные каждого дня хранятся отдельно

### Эмодзи прогресса
| Прогресс | Эмодзи |
|----------|--------|
| 0 | ❌ |
| 1–24 | 🔴 |
| 25–49 | 🟡 |
| 50 (цель) | ✅ |
| 100+ (2x цель) | 🏅 |

## Команды бота

| Команда | Описание |
|---------|----------|
| `/start` | Запустить бота, показать справку |
| `/summary` | Показать текущую сводку |
| `/members` | Список участников |
| `/addmember Имя` | Добавить нового участника |
| `/reset` | Сбросить счётчики (только для админов) |

---

## Добавление нового участника
Прямо в чате напиши:
```
/addmember Новое_Имя
```
Бот добавит его в список и начнёт отслеживать его сообщения.

## Запуск на сервере (чтобы бот работал 24/7)

### Вариант 1: screen (простой)
```bash
screen -S regbot
python bot.py
# Ctrl+A, затем D — выйти из screen, бот продолжит работать
```

### Вариант 2: systemd (надёжнее)
Создай файл `/etc/systemd/system/regbot.service`:
```ini
[Unit]
Description=Registration Bot
After=network.target

[Service]
WorkingDirectory=/путь/к/папке/regbot
ExecStart=/usr/bin/python3 bot.py
Restart=always

[Install]
WantedBy=multi-user.target
```
Затем:
```bash
systemctl enable regbot
systemctl start regbot
```
